# 仿掘金站点简易后端

## 介绍

使用 `json-server` 搭建的完整模拟 REST API 接口



请求方法：

- 数据查询要使用 `GET`
- 新增数据要使用 `POST`
- 删除数据要使用 `DELETE`
- 修改数据使用 `PUT` 和 `PATCH`



## 环境搭建

安装 node

https://nodejs.org/en/

安装 json-server

```shell
npm install -g json-server
```

启动服务

```shell
json-server --watch db.json
```
